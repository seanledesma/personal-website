My Personal Interests

I have always been interested in computers and elctronics. When I was a teenager you could usually find me tinkering
with something. I almost killed the family computer on several occasions because I had downloaded some virus while
tinkering with it. I didn't have my own personal computer, so most of my time was spent on my android phone.
I unlocked the bootloader, rooted it, and started installing custom firmware.

Besides messing around with my computer and cell phone, I've also had a great interest in
video making and editing. I rented video equipment from my High School on several occasions,
using it to create short youtube videos with my friends. I became more and more proficient with
Adobe Premiere Pro, although I don't do much of that these days.

Currently I'm more interested programming and networking. I've been setting up my own personal home server,
I run ProxMox which is a HyperVisor OS, and then I run Debian Linux on top of that in a VM. This way
I'm able to spin up more OS's in VM's if I wanted. Inside Debian I run Docker, which I access from the web
using something called Portainer. From Docker I run a whole bunch of services, such as Pi-Hole, Plex,
NextCloud and Home Assistant.
